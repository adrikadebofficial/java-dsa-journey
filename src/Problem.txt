Given a sentence as a string input, write a program that counts how many times each word appears in it. Then look up a specific word and print its frequency.
Example:
Input sentence : "hello world hello java hello world java hello"
Query word     : "hello"

Output         : 4
Constraints to keep in mind:

Treat the input as case-insensitive ("Hello" and "hello" are the same)
Words are separated by spaces
