You are building a simple phone book application. Implement the following operations:

ADD — Add a contact with a name and phone number
SEARCH — Given a name, return their phone number
DELETE — Remove a contact by name
DISPLAY — Print all contacts in the phone book


Input Format
ADD Alice 98765
ADD Bob 12345
ADD Charlie 55555
SEARCH Alice
SEARCH Dave
DELETE Bob
DISPLAY

Expected Output
Alice added.
Bob added.
Charlie added.
Alice -> 98765
Dave not found.
Bob deleted.
Alice -> 98765
Charlie -> 55555

Constraints

Names are unique (no two contacts have the same name)
Phone numbers are stored as Strings (they can have leading zeros)
SEARCH and DELETE on a non-existent name should print "[name] not found."
